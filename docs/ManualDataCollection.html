Data Collection Procedure Guide

⚠️ Critical Precautions
  • Timing: Data collection must be performed on Sunday, before the Daily Reset Time (GMT+02:00).
  • Deadline: Complete the process before the Weekly Raid ends (GMT+1:00). Once the raid expires, data cannot be retrieved.
  • Optimal Window: To ensure the latest data is captured, it is safest to collect data 10–20 minutes before the raid ends (unless all attacks are already finished).
  • Restrictions: Do not perform any actions other than the specified navigation. Refreshing or closing the browser during the process will result in data loss.

Phase 1: Main Data Collection (Sunday)

  1. Go to Hero Wars Official Site or refresh the page to initialize the game state.
  2. Navigate to:
    Home > Daily Quests > Guild Quests > Status
  3. Navigate to:
    Home > Daily Quests > Guild Quests > Prestige
  4. Navigate to:
    Home > Daily Quests > Guild Quests > Statistics
  5. Navigate to:
    Home > Guild > Guild War > Clash of Worlds > Log
    • Check the two battle records for the current week. Click the "More" button for each record once to confirm details.
  6. Navigate to:
    Home > Guild > Guild War > Guild War > Log
  7. Navigate to:
    Home > Guild > Guild War > Guild War > Leagues > Previous week
    . Select your guild's league and close the in-game window.
  8. Navigate to:
    Home > Guild > Asgard > Guild Raid > Log
  9. Open Tampermonkey > Select "Hero Wars Data Export (Customized)" > Click "Export Data (CSV)".
  10. Save the following three files to a local folder (Organizing by weekly folders is recommended):
    • - stats_<date>.csv
    • - event_log_<date>.csv
    • - boss_log_<date>.csv

Phase 2: Supplementary Statistics Update

This task updates the remaining statistical data (excluding raids) and can be performed at any time from Monday through Saturday (strictly excluding Sundays).

  1. Go to Hero Wars Official Site or refresh the page.
  2. Navigate to:
    Home > Daily Quests > Guild Quests > Statistics
  3. Open Tampermonkey > Select "Hero Wars Data Export (Customized)" > Click "Update Sunday Stats".
  4. Select the stats_<date>.csv file you previously saved.

After completing these steps, final reports will be generated via Google Sheets.
